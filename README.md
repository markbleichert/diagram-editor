# Diagram editor
A proof of concept for a React based diagram editor.


![alt text](img/screenshot.png "Data flow visualization")

## Features

* React 16
* Webpack 4
* Babel 7
* Diagramming editor

## Installation

* `git clone https://github.com/markbleichert/diagram-editor.git`
* cd diagram-editor
* npm install
* npm start
* visit `http://localhost:8080/`
